# 常見 OCR 陷阱

長文、雜訊多、表格多、或者語義可疑時讀呢份 reference。

## 高信心錯字

無括號都要按語境同 PDF 查：

- `禁賭/黎賭`、`重大/重火`、`呈文/呈女`、`查辦/杳辦`、`議决/議次`
- `綱領/網領`、`總綱/總網`、`聯絡/聯落`、`任期內/任期丙`
- `本局/本屈`、`籌/薵`、`膏腴/膏膄`、`盤據/盤拔`
- `投票/投樂`、`中學/巾學`、`提請/捉請`、`法定/已法定`
- `官吏/官火`、`大害/火害`、`股東/股束`、`股票/股樂`

## 官職、人名、套語

- `令/合` 好常錯。查人名、官職、`法令`、`命令`、`致令`；但保留真官文 `合就`、`合行`、`合併`。
- `另/男` 常見於議案官文。查 `另增`、`另籌`、`另案`、`另招商人`；但案情清單入面真 `男女` 要保留。
- `札/礼`、`箚/符/筍`、`堂/常` 要按 `札行`、`札司`、`箚覆`、`督部堂` 等套語判斷。
- `道/逍`、`遣/逍`、`遺/逍` 常見於 `巡警道`、`隧道`、`差遣`、`遺失`、`載道`。
- `議員`、`代表員`、`督憲代表`、`臬憲代表` 可能變 `議負`、`化表員`、`督憲化表`、`臬化表`。
- `已/己/巳` 同 `戌/戍` 要按語法同地支查；`己酉`、`戌` 呢類真干支要保留。
- 人名、地名先查 PDF，唔好靠現代記憶直接改。

## 數字同符號雜訊

- `∵`、`≡`、`1` 常代表中文數字或近形字，但唔可以全局替換。
- 查 `第十∵欵`、`其∵曰`、`二≡六五`、`八百九十∵名`、`∵十名`。
- `1` 喺套語入面可能係 `壽`、`諮`、`議`、`墨`、`麟`、`窒`、`姑`，要逐處看圖。
- 金額單位要按語境：`萬員`、`月費壹員` 可能係 `萬元`、`月費壹元`；`議員`、`敎員`、`每員` 係人員。
- 財政/稅務議案常有高信心錯：查 `收入/收人`、`每年/錄年`、`香燭/香惱`、`元寶/寳`、`土造/圡造`、`實業/寳業`、`殖業/植業`。

## 清單同條文

- 章程同長議案要查項號連續性。
- OCR 常漏新 item 前嘅 `-`，或者將新 item 錯縮入上一 item。
- 縮入行以 `一...`、`甲...`、`乙...` 開頭時，要查 PDF 判斷係續文定新項。
- `第一條` 等通常係 bullet item；`附說` 同跨頁續文若版面支持，縮入同一 item。
- OCR 亦會將正文誤當 bullet；若 bullet 令文義斷裂，先嘗試還原成正文。
- `原草一二`、`修正一五`、`說明一`、`原草一八` 呢類草案比較標籤可能係真結構，唔好未睇 PDF 就刪或重編號。

## 分段同 heading

- OCR 行尾語法未完要合併；例如結尾係 `第`、`於十`、`未`、`以`、`由`、`可請`、`業經`。
- 唔好只因兩行都長就合併；奏摺/呈文/請願入面 `督部堂`、`旨`、`上諭`、`皇上` 可能係版面抬頭。
- 速記錄發言人轉換係強分段訊號：`某號議員謂`、`督部堂代表謂`、`議長謂`。
- `箚復爲箚復事...` 呢類黐住嘅標題要拆成 heading 加正文；但唔好順手統一全篇 `箚/劄/剳/札`。
- 同頁可能交錯多份短文件；頁頂短句可能係新案，唔一定係上一段續文。
- 合訂講義以課目作主要結構，例如 `法學通論`、`政治學`、`憲法講義`、`行政法學`、`大淸會典要義`、`經濟學`。
- 刪 `<!-- page_... -->` 後要查相鄰普通正文行，避免留下 Markdown soft wrap。

## 有標點講義

- 有啲 PDF 本身有舊式標點，唔係所有歷史文本都無標點。
- 保留 `。`、`。。`、`、`；唔好將 `。。` 自動正規化成 `。`。
- 法律/政治/經濟詞源段可能有合法拉丁詞：`Politike`、`Polis`、`jus`、`law`、`droit`、`Recht`、`diritto`、`yu`、`lagu`、`lex`、`directum`。
- Japanese-style 法律/經濟借詞、音譯、貨幣名、人名要按 PDF 查，唔好現代化。
- 最後非空白頁若印刷本身停喺半句，而下一頁係空白或另一篇，保留半句並報告，唔好補完。

## 尾頁同視覺陷阱

- 有印章、反白、修補塊、淡字嘅尾頁，可能將兩段黐埋；要睇普通圖、反色圖、crop。
- OCR 空白 notice 可能漏規則、表格、尾頁小字、圖。
- Apple/macOS OCR 同 OCR JSON 對直排講義有用，但會漏淡字、誤判空白、拆亂表格；只當提示。
- 最後頁可能係另一題名或附錄；未確認屬當前 Markdown 前唔好硬併。
- 半頁版面有大空白唔代表漏字；按 PDF 查欄位。
- 日期或損壞呈文讀唔清時，保留最有根據讀法並報告 uncertainty，唔好由年代猜。

刪 page comments 後查 soft wrap：

```bash
python3 - <<'PY'
from pathlib import Path
lines = Path("TARGET.md").read_text().splitlines()
def structural(s):
    t = s.strip()
    return not t or t.startswith(("#", "- ", "|", ">", "```")) or s.startswith("  ")
print(sum(1 for i in range(len(lines)-1) if not structural(lines[i]) and not structural(lines[i+1])))
PY
```
